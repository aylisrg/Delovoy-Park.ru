## Технические требования

- **Формат**: JPEG для фотографии, SVG для логотипа, WebP по возможности
- **Размеры**: Hero — минимум 1920×1080px, карточки — 800×600px
- **Качество JPEG**: 80-85% (баланс качество/размер)
- **Оптимизация**: Использовать `next/image` для автоматической оптимизации

## Инструкция по добавлению изображений

1. Скачайте изображения с текущего сайта https://delovoy-park.ru/
2. Оптимизируйте через TinyPNG или аналог
3. Назовите файлы согласно таблицам выше
4. Разместите в `/public/images/`
5. Обновите компоненты для использования `next/image`
6. Обновите статус в этой таблице

## Использование в коде

```tsx
import Image from 'next/image';

// Пример использования
<Image
  src="/images/hero-exterior.jpg"
  alt="Фасад бизнес-центра Деловой Парк"
  width={1920}
  height={1080}
  priority={true}
  className="object-cover w-full h-full"
/>
```
